#### THIRDREALITY Zigbee Smart Color Bulb ZL1 - Factory Reset
---
Power the smart color bulb, power cycle(power off and on) smart color bulb 5 times in a row to factory reset it, it flashes warm white - cold white - red - green - blue in a row within 3 seconds, then it turns solid warm white, indicating it enters Zigbee pairing mode. It will exit Zigbee pairing mode if not paired within 3 min.


#### Ikea TRADFRI light bulb - Factory Reset
